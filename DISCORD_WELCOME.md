## 1) Welcome DM (Welcomer → DM)

```
Hey, welcome to the SynapseKit community! 👋

A few things to get started:
• Head to #roles and pick what you're building
• Introduce yourself in #introductions
• If you need help, drop a message in the relevant #*-help channel

Docs: https://synapsekit.github.io/synapsekit-docs/
GitHub: https://github.com/SynapseKit/SynapseKit

Glad to have you here.
```

---

## 2) Auto-assign Member role (Autoroles)

- Role: **Member**
- Trigger: on join

---

## 3) Welcome message in #introductions (Welcomer → Channel)

```
Welcome {user}! Tell us what you're building 🛠
```

---

## 4) #rules pinned message (keep it short)

- Be respectful
- Keep questions in the right help channel
- No spam or self-promo without context
- English preferred (or include a translation)
- No DM-ing members without permission

---

## Notes

- Set up channel structure first (see COMMUNITY.md / DISCORD_ROLES.md).
- If you change channel names, update the DM text accordingly.
